# Privacy Policy

Effective date: 2026-02-10

CPL Enhancer is a browser extension for CPLManager. It does not collect personal data,
track browsing history, or send analytics. All processing happens locally in the browser.

## Data We Process
- Settings and thresholds you choose in the Options page.
- These preferences are stored with `chrome.storage.sync` so they follow your Chrome profile.

## Data Sharing
- No data is sold or shared with third parties.
- No data is transmitted to external servers by the extension.

## Permissions
- `storage`: save your settings and thresholds.
- `activeTab`: apply actions only on the active CPLManager tab.
- `scripting`: inject UI enhancements on CPLManager.
- Host permissions: run only on `https://cplmanager.com/*` and `https://www.cplmanager.com/*`.

## Remote Code
The extension does not load or execute remote code. All scripts are packaged in the extension.

## Changes
If this policy changes, the updated version will be included in the repository and store listing.
